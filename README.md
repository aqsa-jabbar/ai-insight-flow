🔎 AI-Powered Research & Report Generator
Give it a website URL or a platform/company name, and it generates a structured research report, then lets you ask follow-up questions about that same report. No memory is kept between different links/topics, and no RAG/vector database is used - each report is generated fresh from live web search (Serper.dev) and website scraping.
